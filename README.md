# 程序介绍

## 环境
python = 3.7

pytorch=1.8

cuda = 11.1

torchvision = 0.9

numpy, matplotlib, opencv, h5py

## 文件结构
BP网络：work1BP.py

RBF网络：work1BP.py

ResNet网络：work1resnet.py, MyDatasetCat.py

## 文件夹介绍
./dataset_h5：存放猫与非猫的h5数据集

./internet_image：网络截图，用于测试网络

./results：保存训练结果曲线，其中FN、FP、TN、TP为resnet网络分类结果

./weights：存放网络的训练权重结果
